## How to Learn From Solutions
Verify allocation and deallocation points. Make sure every `new` has a matching
`delete` or `delete[]`.

## Common Mistakes
- Dereferencing invalid pointers
- Using mismatched delete operations

## Mini-FAQ
**Q: Why not use vectors?**
