# Examples

Structs and classes used in simple DSA models.

## How to Use
Read each class definition, then run the file and trace object behavior.

## Files
- `example_01.cpp` - Simple linked list node
- `example_02.cpp` - Stack class using vector
- `example_03.cpp` - Rectangle class with methods
- `example_04.cpp` - Student struct with helper method

## Build and Run
```bash
g++ example_01.cpp -o example_01
./example_01
```

## What to Observe
- Constructor usage and initialization
- Encapsulation of data and methods
- Simple data structure modeling

## Common Mistakes
- Leaving members uninitialized
- Exposing internal fields unnecessarily

## Mini-FAQ
**Q: Why use classes in DSA?**
A: They model nodes and structures cleanly and safely.
