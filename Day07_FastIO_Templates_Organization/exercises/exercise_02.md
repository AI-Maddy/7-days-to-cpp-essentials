# Exercise 02: Template Min of Three

## Task
Create a function template that returns the minimum of three values.

## Input
Three values `a`, `b`, and `c` (same type).

## Output
The minimum value.

## Example
Input:
```
5 2 9
```
Output:
```
2
```

## Notes
Use comparisons and return the smallest value.

## Concept Focus
- Template comparisons
- Returning a single value

## Constraints (Suggested)
- Values are comparable with `<`

## Approach
Compute the minimum of three values using comparisons.

## Edge Cases
- All values equal
