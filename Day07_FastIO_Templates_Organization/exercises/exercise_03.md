# Exercise 03: Fast I/O Sum

## Task
Read `n` integers and output their sum using fast I/O settings.

## Input
- Integer `n`
- `n` integers

## Output
The sum of the numbers.

## Example
Input:
```
5
1 2 3 4 5
```
Output:
```
15
```

## Notes
Use `ios::sync_with_stdio(false);` and `cin.tie(nullptr);`.

## Concept Focus
- Fast I/O setup
- Summation loop

## Constraints (Suggested)
- $1 <= n <= 10^6$

## Approach
Enable fast I/O, read all values, sum, and print.

## Edge Cases
- Large `n` values
